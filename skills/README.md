# Skills

This directory contains installable Agent Skills. Each child folder is a complete skill package and must be copied as a folder, not as a single `SKILL.md` file.

| Skill | Description |
| --- | --- |
| [`brainstorm`](brainstorm/SKILL.md) | Focused back-and-forth thinking mode for exploring options, making decisions, and converging on a clear direction. |
| [`illustrations`](illustrations/SKILL.md) | Generates Ian-style English article illustrations using the Xiaohei visual IP and supporting style references. |
| [`setup-workspace`](setup-workspace/SKILL.md) | Interviews a knowledge worker and scaffolds a portable AI-agent-friendly workspace. |
| [`to-prd`](to-prd/SKILL.md) | Synthesizes current conversation and codebase context into a PRD for the project issue tracker. |

Agents should read the repository root [`skills.json`](../skills.json) for machine-readable discovery data.
