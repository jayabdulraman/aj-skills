# Preset: Enrollment Coach

## When to use
Role is enrollment coach/counselor, admissions/recruitment advisor. Guides prospective students from inquiry → application → enrollment; manages a pipeline; runs outreach cadences.

## Suggested structure
```
pipeline/                 # by stage or term (inquiry, applied, admitted, enrolled)
  outreach-logs/  follow-ups/  notes/
playbooks/                # scripts, FAQs, objection handling, program info
reporting/                # funnel metrics, conversion
templates/  work-memory/  connectors/  resources/  archive/
```

## Recommended templates
outreach-sequence · call-script · faq-objection-handling · prospect-notes · follow-up-message · funnel-report

## AGENTS.md role snippet
> Focus areas: prospect engagement, guiding the enrollment journey, timely follow-up, conversion. Delegate: drafting outreach sequences and scripts, building FAQs, formatting notes, summarizing funnel trends. **Prospect data is sensitive** — keep identifiable info out of shared/synced outputs; the CRM is the system of record.

## Tailoring notes
Ask their CRM (system of record — keep prospect data there), dialer/comms tools, and program catalog. Bake program facts and approved messaging into `playbooks/`. Default to privacy; this workspace holds playbooks and de-identified summaries.
