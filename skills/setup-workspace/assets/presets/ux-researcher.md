# Preset: UX Researcher

## When to use
Role is UX/design researcher, user researcher, research ops. Runs studies, interviews, usability tests, surveys; synthesizes findings; maintains a research repository.

## Suggested structure
```
studies/                  # one folder per study
  <study>/
    plan/  guide/  recruiting/  raw/ (transcripts, recordings, notes)  analysis/  findings/  readout/
repository/               # cross-study insights, personas, atomic findings
templates/  work-memory/  connectors/  resources/  archive/
```

## Recommended templates
research-plan · interview-guide · usability-test-plan · survey · synthesis/affinity · findings-report · readout-one-pager

## AGENTS.md role snippet
> Focus areas: study design, recruiting, moderation, synthesis, sharing insight. Delegate: transcript cleanup, note formatting, drafting guides/screeners, tagging findings, building readouts. Keep raw participant data private; never expose PII in shared outputs. Prefer evidence-linked claims (quote → finding → recommendation).

## Tailoring notes
Ask which research repo/tool they use (Dovetail, Notion, etc.) and meeting-notes tool. Set up a clear `raw/` vs `findings/` split so sensitive data stays separate from shareable insight. Add a personas/segments doc in `repository/`.
