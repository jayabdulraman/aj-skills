# Preset: Instructional Designer

## When to use
Role is instructional designer, learning experience designer, curriculum developer. Produces courses, modules, assessments, rubrics; works in an LMS; collaborates with faculty/SMEs.

## Suggested structure
```
courses/                  # one folder per course
  <course>/
    design-doc/  outline/  modules/  assessments/  rubrics/  media/  sme-notes/  qa-review/
standards/                # design standards, QM/accessibility checklists
templates/  work-memory/  connectors/  resources/  archive/
```
(Artifact-first variant works too if they manage many courses producing the same artifacts.)

## Recommended templates
course-design-doc · module-outline · learning-objectives · assessment-spec · rubric · qa-accessibility-checklist

## AGENTS.md role snippet
> Focus areas: learning design, SME collaboration, assessment/rubric quality, accessibility & quality standards. Delegate: outlines, drafting objectives, formatting assessments/rubrics, QA checklists. Apply accessibility (WCAG) and any course-quality framework (e.g. QM) by default.

## Tailoring notes
Ask their LMS (Canvas, etc.), doc home, and any quality framework (Quality Matters, institutional standards) to bake into `standards/`. If they build branded learner-facing materials, suggest pairing with the org's design-system skill.
