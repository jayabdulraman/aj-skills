11 · Rules

Don't

Constraints that keep EdPlus visually coherent with ASU.

Don't use maroon as a background color.

Use white, Gray 7, or Gray 1 for content emphasis. Maroon belongs on primary buttons and brand accents.

Don't use gold text on a maroon background.

That color combination is undesirable. Keep gold for highlights, eyebrow chips, and on dark or white surfaces.

Don't color icons maroon.

Icons default to Gray 1 (#191919). State icons use their state color (info blue, success green, warning orange, error red).

Don't use all caps in body or UI text.

Sentence case only. The one exception is abbreviations (ASU, UDS, PDF) and proper names.

Don't round card corners.

Cards, inputs, modals, tooltips, and eyebrow chips all use border-radius: 0.

Don't use italics.

Emphasize with bold, layout, or sentence structure — never italics.
