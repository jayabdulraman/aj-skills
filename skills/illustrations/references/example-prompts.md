```text
Use $illustrations, but do not generate images yet.
Analyze the article below and propose a shot list of about 5 inline illustrations.
For each image, include placement, theme, core meaning, structure type, what Xiaohei is doing, suggested elements, and short English handwritten labels.

```

## Generate Article Illustrations

```text
Use $illustrations to generate 4 clean, absurd Xiaohei-style inline illustrations for the article below.
Requirements: 16:9 horizontal PNGs, pure white background, black hand-drawn line art, sparse red/orange/blue English handwritten labels.
Each image should explain one core structure. Avoid PPT infographics, cute cartoons, dense diagrams, and top-left type titles.

<paste article>
```

## Longform Illustration Strategy

```text
Use $illustrations to create an illustration strategy for this long article.
Do not distribute images evenly. Choose cognitive anchors such as core claims, input-output loops, before/after contrasts, common pitfalls, handoff paths, or role-state changes.
Default to 6-8 images. Output the shot list first and do not generate images yet.

<paste article>
```
